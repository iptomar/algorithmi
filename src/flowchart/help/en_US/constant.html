Constantes

Uma constante define um símbolo cujo valor permanece inalterável durante o seu ciclo de vida. Segue as mesmas regras que a definição de variáveis excepto que, não é possível omitir o valor de inicialização.

Sintaxe geral:

constante [tipo] [nome] <- [valor]

constante [tipo] [nome] <- [expressão]

constante [tipo] [nome] <- [valor],[nome] <- [expressão]

Exemplos:

constante inteiro meses <- 12

constante real pi <- 3.14
